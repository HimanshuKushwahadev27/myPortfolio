infra-core & infra-spring-boot · Published on Maven Central

A framework-agnostic Java library for distributed infrastructure patterns, with a Spring Boot starter that exposes them through constructor injection.

  • Implements rate limiting, idempotency, caching, and atomic stock management backed by Redis, with clean interfaces and a factory pattern for zero-framework dependency.
  • Ships with infra-spring-boot auto-configuration so the same primitives are available in Spring Boot applications without boilerplate.
